# DevCamper API

> Backend API for DevCamper app, which is a bootcamp directory website

## Usage

Rename "config/config.env.env" to "config/config.env" and update values/settings to your own

## Install Dependecies

```
npm install
```

## Run App

```

# Run in developer mode

npm run dev

# Run in production mode

npm start
```
